- **所有权**  
  所有权是一组用于 rust 程序管理内存的规则:
  1. rust 中每个值在任何时候都有且仅有一个所有者.
  2. 当所有者离开它所在的作用域时,它所拥有的值将被回收.

* 结合**浅拷贝**(只拷贝栈上数据)和**深拷贝**(堆和栈上的数据都拷贝)理解 rust 的**移动语义**和**克隆语义**.  
  一个类型实现了 Copy trait，那么一个旧的变量在将其赋值给其他变量后仍然可用.  
  rust 中基础类型都实现了 Copy trait.
  Rust 不允许自身或其任何部分实现了 Drop trait 的类型使用 Copy trait.

- **引用与借用**
  1. 是否可变(决定了借用者是否拥有该引用的所有权)
  2. 使用&符号来创建一个引用
  3. 创建一个引用的行为称为**借用**

* 引用的规则:
  1. 在任意给定时间，要么 只能有一个可变引用，要么 只能有多个不可变引用.
  2. 引用必须总是有效的.

- **String 类型**

  1. 初始化: String::from("abc");

- **Slice 类型**

  1. 引用集合中一段连续的元素序列, 它没有所有权.
  2. 示例: let s = String.from("xxxxxxx"); let si = &s[1,3];

- **trait 类型[自定义类型]**
