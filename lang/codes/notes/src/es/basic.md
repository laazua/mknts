# 基本概念

- **cluster**  
  群集是一个或多个节点（服务器）的集合，这些节点共同保存整个数据，并提供跨所有节点的联合索引和搜索功能.
- **node**  
  节点是属于群集的单个服务器，存储数据并参与群集的索引和搜索功能.
- **index**  
  索引是具有某种相似特征的文档的集合.
- **document**  
  文档是可以编制索引的基本信息单元.
- **shareds&replicas**  
  分片和副本数是为了解决索引所存储的数据超过单个节点的硬件限制,提供了将索引细分为多个部分（称为分片）的功能.  
  创建索引时，只需定义所需的分片数即可。每个分片本身就是一个功能齐全且独立的“索引”，可以托管在集群中的任何节点上.
