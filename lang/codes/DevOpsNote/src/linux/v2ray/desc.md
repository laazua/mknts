* *说明*
1. v2ray为服务端+客户端模式
2. 将服务端部署在境外服务器上(v2ray run -c server.json)
3. 将客户端部署在国内服务器上(v2ray run -c client.json)
4. 对于windows系统下载v2rayN客户端软件进行配置
5. linux系统配置:
   export http_proxy="http://192.168.165.89:8080"
   export https_proxy="http://192.168.165.89:8080"
