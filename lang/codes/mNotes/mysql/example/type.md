###### mysql数据类型


* **数据类型**
```
    -- 数值类型
        TINYINT, SMALLINT, MEDIUMINT, INT, BIGINT, FLOAT, DOUBLE, DECIMAL
    -- 时间类型
        YEAR, TIME, DATE, DATETIME, TIMESTAMP
    -- 字符串类型
        CHAR, VARCHAR, BINARY, VARBINARY, BLOB, TEXT, ENUM, SET

    -- 注: 为了优化存储,提高数据库性能,任何情况下均应该使用最精确的类型来存储对应的值
```
